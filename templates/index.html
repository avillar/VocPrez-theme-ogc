Welcome to the OGC Community Application Profiles Server

This is a research tool based on the OGC Definitions Server to publish community sourced profiles of OGC resources.

The OGC Definitions Server publishes a range of resources managed by the OGC, and identified by unique stable web addresses (IRIs or URIs). These are determined through a set of policies managed by the OGC Naming Authority and managed through normal operational processes.

The knowledge graph behind the definitions server is flexible and extensible, the research tool supports publication of a range of related resources needed to describe interoperable data models.

Definitions are made available in an interoperable form using the SKOS vocabulary (or "taxonomy") model in both human- and machine-readable forms.

This repository supports richer information models based on the OWL language and the W3C Profiles Vocabulary, with alternative profiles of the content available to support different implementation platforms for these models, such as JSON-LD

The "alternate profiles" will provide a complete list of the available resources per definition.

See the Definitions Server About Page for more information about the OGC Definitions Server.
